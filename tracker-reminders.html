You can find two sections on this page:

  1. Code PR author reminders: docs PRs waiting on a code PR author's review or reply, grouped by name.
  2. Escalated to the Core Team: a shared queue of PRs formally sent to the Core Team for an opinion and review.

Using this list:

  • Use the links above to jump straight to a section — Core Team, section 2 is your queue.
  • Under Code PR author reminders, click your name to jump to your own items.
  • Check a box to track your progress — it saves to your own browser only. The list updates when the tracker runs on schedule.

To review a PR:

  1. Open the PR's Files changed tab, then use Review changes → Submit review to approve it or request changes.
  2. To comment on specific code, hover over the line and click the blue + that appears; for several lines, click and drag across them. Type your note, click Start a review, and repeat for other lines — then Submit review when you're done.
