The three bands — whose turn is it?

  • Need you today — actions only you can take.
  • Waiting on others — you’ve done your part; a clock is running.
  • Monitoring — the author replied and you’ve already looked; a normal back-and-forth is running, nothing to do.

The day clock

The counter on the right of each row counts days since your reminder, with two deadlines — and it stops the moment the author replies (the row then moves to “Need you today” so you can check their response):

reminder sent day 7 · follow up day 14 · escalate

Reading a row

  • Coloured left edge — how urgent the row is: red overdue → orange due soon → blue actionable → grey triage.
  • Open Pills — the rounded tags next to a PR number. Facts about that PR: Draft / Open / Merged / Closed.
  • Review this docs PR Chips — the rectangular buttons at the bottom of a row. Actions for you to take; the colour tells you the kind of task (see colour key).

The two labels

pending-pr-merge — removed once the code PR merges · needs-backport — added when a PR targets an older branch than the latest.
